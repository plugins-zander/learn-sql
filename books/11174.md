# 命令

```sql
语法格式：
DROP TABLE [ database_name . [ schema_name ] . | schema_name . ]
        table_name [ ,...n ] [ ; ]
其中，table_name是要被删除的表名。
例如，要删除表XSB，使用的T-SQL语句为：
USE PXSCJ
GO
DROP TABLE XSB
```
