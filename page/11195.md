# 关系比较运算符

　　比较运算符用于测试两个表达式的值之间的关系，这种关系是指等于、大于、小于、大于等于、小于等于、不等于、不小于、不大于等。比较运算符几乎适用于所有的表达式（除了text、ntext 或image 数据类型的表达式外）。表列出了Transact-SQL 支持的比较运算符。

 

比较运算符 

| 运算符 | 含义       |
| ------ | ---------- |
| =      | 等于       |
| >      | 大于       |
| <      | 小于       |
| >=     | 大于或等于 |
| <=     | 小于或等于 |
| <>     | 不等于     |
| !=     | 不等于     |
| !<     | 不小于     |
| !>     | 不大于     |
