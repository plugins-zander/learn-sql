# 位操作

 位运算符

　　位运算符是表示在两个操作数之间执行按位进行运算的符号，操作数必须为整型数据类型之一，如bit、tinyint、smallint、int、bigint等，还可以是二进制数据类型（image 数据类型除外）。下表列出了位运算符及其含义。

位运算符及其含义

| 位运算符 | 含义                     |
| -------- | ------------------------ |
| &        | 对两个操作数按位逻辑与   |
| \|       | 对两个操作数按逻辑位或   |
| ^        | 对两个操作数按位逻辑异或 |
| ~        | 对一个操作数按位逻辑取非 |
